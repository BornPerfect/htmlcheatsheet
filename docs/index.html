HTML Cheatsheet

Document Metadata

Tag Name Description
<head> Head The head element contains machine-readable information (metadata) about the document, like its title, scripts, and style sheets.
<link> Link The Link element specifies relationships between current document and external resource. It is used to link style sheets.
<meta> Meta The Meta element represents metadata that cannot be represented by other HTML meta-related elements.
<style> Style The Style element contains style information for a document, or a part of it.
<title> Title The Title element defines the document's title that is shown on the browser's title bar or a page's tab.


Content Sectioning

Tag Name Description
<footer> Footer The Footer Element represents a portion of a document whose content is only indirectly related to the document's main content.
<header> Header The Header element represents introductory content. There are six levels of headings, written as <h1-h6>
<main> Main The Main element represents the dominant section of the body of a document. It only consists of content directly related to the central idea of a topic.
<section> Section The Section element represents a standalone section, which doesn't have a more specific semantic element to represent it.
<div> Division The Division element is the generic container for flow content. It has no effect on the layout until styled by css.


Text Content

Tag Name Description
<ul> Unordered List The Unordered List element represents a list of unordered items, typically rendered as a bulleted list.
<ol> Ordered List The Ordered List element represents a list of Ordered items, typically rendered as a numbered list.
<p> Paragraph The Paragraph element represents a Paragraph.
<li> List Item The List Item element is used to represent an item in a list.


Inline Text Semantics

Tag Name Description
<a> Anchor The Anchor element, with it's href attribute, creates a hyperlink to web pages, files, email addresses, locations on the same page, or any URL.
<br> Line Break The Line Break element produces a line break in text, useful for writing poem's or addresses, where Division of lines are significant.
<em> Emphasis The Emphasis element marks text that has stress emphasis. It is typically Rendered as italics.
<strong> Bold/Importance The Bold/Importance element indicates it's contents have a strong, importance, seriousness or urgency. It is typically rendered as bold text.


Images and Multimedia

Tag Name Description
<audio> Audio The Audio element is used to embed sound content in documents, represented using the src attribute.
<img> Image The Image element embeds and image into the document.
<video> Video The Video element embeds a media player which supports video playback into the document.
